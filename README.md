# ConnectSphere

A full-stack MERN application that helps students collaborate on projects, build teams, and discover internship opportunities.

## Features

- User Authentication (JWT)
- Student Profiles
- Project Management
- Internship & Job Listings
- Team Collaboration
- Dashboard
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JSON Web Token (JWT)

### Other Services
- Cloudinary
- Nodemailer

## Project Structure

```
client/
server/
```

## Installation

### Clone the repository

```bash
git clone https://github.com/29atharvvaaa/ConnectSphere.git
```

### Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### Run the project

Frontend

```bash
npm run dev
```

Backend

```bash
npm run dev
```
